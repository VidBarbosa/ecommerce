== Description ==

This is a plugin to connect external APIS with a focus on e-commerce and to bring these products and functionalities to Wordpress and Woocommerce, a custom solution for Etraining. 

== Installation ==

Upload the Ecommerce plugin to your store, activate it, and then enter your API external.

1, 2, 3: You're done!
